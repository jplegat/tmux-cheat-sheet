Tmux Cheat Sheet

A terminal multiplexer that lets you run many windows inside a single terminal buffer, split into panes and controlled through one keyboard.

Prefix key: Ctrl + B
Attach & Detach
$ tmux Start a new tmux session
$ tmux attach Attach to a session running in the background
Ctrl+B thend
Detach from session, leaving it running in the background
Ctrl+B then&
Exit and quit tmux
Ctrl+B then?
List all key bindings (press Q to exit)
Window Management
Ctrl+B thenC
Create new window
Ctrl+B thenN
Move to next window
Ctrl+B thenP
Move to previous window
Ctrl+B thenL
Move to last window
Ctrl+B then0–9
Move to window by index
Session Management
Ctrl+B then)
Move to next session
Ctrl+B then(
Move to previous session
Ctrl+B then Ctrl+Z
Suspend session
Split Window into Panes
Ctrl+B then%
Vertical split — panes side by side
Ctrl+B then"
Horizontal split — one pane below the other
Ctrl+B thenO
Move to other pane
Ctrl+B then!
Remove all panes but the current one
Ctrl+B thenQ
Display pane index numbers
Ctrl+B then Ctrl+↑ / ↓
Resize current pane north / south
Ctrl+B then Ctrl+← / →
Resize current pane west / east
Multiplex
Ctrl+B then:
Access the tmux command prompt
:setw synchronize-panes on Synchronize panes — send a command to many hosts at once
